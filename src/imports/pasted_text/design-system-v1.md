The core idea
Use the site mostly in dark mode, driven by:
rich charcoal backgrounds
grey body text
black and white photography
sparse, intentional use of red and white
blue as a grounding brand color, not a loud CTA color
That gives you a site that feels:
masculine
disciplined
premium
editorial
aggressive without being chaotic
And the black-and-white-to-color hover idea is excellent, especially for this brand. It makes color feel earned, not sprayed everywhere.
Refined visual system
Base environment
This should be the everyday world of the site:
Background: rich charcoal
Surface blocks: slightly lifted charcoal / near-black
Body text: medium-light grey
Photography: mostly black and white
Accent usage: controlled
That lets white and red do real work.
Color roles
You were exactly right that the colors need jobs.
Here is the cleanest system:
White = primary action / forward motion / main CTA
 Use white for the most important action you want people to take right now.
Examples:
Start Training
Apply Now
Book a Consultation
Join Brits Brothers
White will hit hardest in dark mode because it is the sharpest contrast.
Red = urgency / commitment / conversion moments / hard push CTA
 Use red when the action has more heat, pressure, or seriousness.
Examples:
Limited Spaces
Enquire Now
Claim Your Spot
Start Your Transformation
Red should feel like a signal, not decoration.
Blue = structure / heritage / navigation / supporting identity
 Use blue as a brand stabilizer, not the main conversion button color.
Best uses:
navigation hovers
secondary button borders
section backgrounds
quotes
stat dividers
icon backgrounds
subtle overlays
That keeps blue feeling rich and British rather than muddy or lost in the dark palette.

Typography system
This is also a strong call.
You want:
thick
blocky
angular
masculine
paired with an Old English display face used carefully
That’s exactly right. The trick is assigning each one a job.
Font jobs
Primary display font: blocky, heavy, angular
 This should handle:
hero headlines
section titles
pricing titles
CTAs
stats
nav emphasis
This is your main brand voice.
Look for something in the territory of:
wide or condensed athletic grotesks
squared-off sans serifs
industrial display fonts
You mentioned something like Work Sans for body copy, which is fine for the supporting role, but not as the main display font.
Old English display font
This should be used like a signature, not a paragraph font.
Best uses:
short words
stamps
section overlines
initials
accent headings
quotes
“Brits Brothers” moments
badges or labels
Do not use it for:
body copy
long headlines
buttons
pricing details
This font says:
 heritage, identity, code, banner
The block font says:
 power, action, structure
Body font
A clean sans like Work Sans can work well here.
I’d use it for:
paragraphs
testimonials
card descriptions
pricing details
supporting UI text
And yes, in a dark-mode system with grey text, you want body copy to be:
medium weight or semi-bold
slightly more open in spacing
not too thin
Thin grey text on dark backgrounds will die fast.
Photography direction
This is maybe the strongest part of the concept.
Default photography style
black and white
high contrast
grain welcome
sweat, metal, plates, tape, chalk, hands, stance, posture
real texture
shadow-heavy
avoid glossy commercial fitness photography
Hover-to-color
This is a very good move if used selectively.
Best use cases:
coach cards
transformation cards
gallery grid
service tiles
homepage featured images
The effect:
image appears black and white at rest
on hover, full color reveals
maybe red/blue overlay elements appear with it
This makes color feel dramatic and controlled.
It also ties perfectly into your “rips and tears” system because the site can feel mostly restrained until interaction reveals intensity.
Section styling direction
With the torn/splatter dividers plus the dark mode and black-and-white photography, you want rhythm.
A good rhythm would be:
charcoal hero
ripped divider into near-black section
blue-toned identity section
charcoal results section
red-accent CTA strip
off-white highlights only in key callout modules
dark footer
The tears become a branding device, but the content blocks stay clean and structured.
That contrast is what makes the site feel expensive:
 raw edges + disciplined layout

Updated brand design language
Here’s the cleanest version of the system:
Mood:
 British, raw, elite, disciplined, masculine, editorial
Visual tension:
 block structure vs torn edges
 black-and-white imagery vs selective color
 charcoal environment vs sharp white CTAs
 heritage type vs modern sans serif system
Color behavior:
 white = primary action
 red = pressure / urgency
 blue = heritage / support / structure
Typography behavior:
 block font = action and authority
 Old English = identity and heritage
 Work Sans = readability and support
